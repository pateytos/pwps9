About project

The primary goal of this project is to create a system for data governance in order to help organizations to manage their data and comply with General Data Protection Regulation (GRPD) and other regulations by creating a system to store, visualize and manage metadata about the organization data.
Project is conducted within "TIE-13106 2018-01 Project Work on Pervasive Systems" course in cooperation with "Intopalo".
